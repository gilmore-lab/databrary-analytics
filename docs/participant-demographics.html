6 Participant demographics

6.1 Purpose

To summarize the demographic characteristics of participants in shared volumes on Databrary.

6.2 Load data

6.2.1 Demographic data 

library(targets)
targets::tar_load(volume_demog_df, store = "../_targets")
demo_df <- volume_demog_df

The session-level CSV data are stored in a local directory that is not synched with GitHub. To generate the report, one must generate and save the data locally.

6.3 Summary results

6.3.1 Overall

Databrary has demographic data for ~\(n=\) 9777 individual participant-sessions. This number is an underestimate because the number of unshared volumes is 3-4x the number of shared volumes.

demo_df <- volume_demog_df %>%
  dplyr::mutate(., vol_url = paste0("https://nyu.databrary.org/volume/", as.numeric(vol_id)))
